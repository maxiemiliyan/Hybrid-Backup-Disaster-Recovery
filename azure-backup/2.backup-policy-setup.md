##  Step-by-Step: Setup Backup Policy

### 1. Open the MARS Agent Console
- On your Windows Server VM, open **Microsoft Azure Backup**.

### 2. Click on **Schedule Backup**
- This will launch the **Backup Schedule Wizard**.

### 3. Select Items to Back Up
- Choose the specific **files and folders** on the VM that you want to back up.

### 4. Set the Backup Schedule
- Choose how often backups run (e.g., **daily at 9 PM**).
- You can also configure multiple schedules if needed.

### 5. Configure Retention Policy
- Set how long to keep:
  - **Daily backups** (e.g., 30 days)
  - **Weekly backups** (e.g., 12 weeks)
  - **Monthly backups** (e.g., 6 months)
  - **Yearly backups** (e.g., 3 years)


### 6. Choose Data Transfer Settings
- You can set **throttling** (limit backup speed during business hours) if needed.

### 7. Review and Finish
- Review your settings and click **Finish**.
- Initial backup will run based on your schedule or can be started manually.
