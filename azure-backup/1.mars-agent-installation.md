This guide explains how to install and configure the **Microsoft Azure Recovery Services (MARS)** agent inside your Windows Server 2019 virtual machine.

---

##  Prerequisites

- You must have already created a **Recovery Services Vault** in Azure.
- You need to download:
  - **MARS Agent Installer**
  - **Vault Credentials File** from the Azure Portal

---

##  Step-by-Step Installation

### 1. Go to Azure Portal
- Navigate to your **Recovery Services Vault**.
- Select **Backup** from the left menu.
- Choose the **"On-premises"** option and select **"Windows"** for workload.
- Click **"Prepare Infrastructure"**.

### 2. Download Required Files
- **Download the MARS Agent**.
- **Download Vault Credentials** â€” this file is needed to register the VM with Azure.

### 3. Install the MARS Agent on Your VM
- Run the downloaded installer.
- Accept the license terms and click **Install**.

### 4. Register the Server
- Once installed, the **Microsoft Azure Backup** console opens.
- Click **Register Server**.
- Browse and select the **Vault Credentials file**.
- Choose:
  - **Encryption Passphrase** (Save this securely â€” needed for restores)
- Click **Register** to complete.

---

##  Outcome

Once registered, your on-prem VM is connected to Azure Backup and is ready for backup policy configuration.
