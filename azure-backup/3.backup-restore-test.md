##  Step-by-Step: Restore from Backup

### 1. Open the MARS Agent Console
- On the VM, open **Microsoft Azure Backup**.

### 2. Click on **Recover Data**

### 3. Choose Recovery Mode
- Select **"This Server"** (if restoring to same VM).
- Click **Next**.

### 4. Select the Recovery Point
- Choose a **date and time** of the backup from which you want to restore.
- MARS Agent will load the list of available items.

### 5. Browse and Select Files/Folders
- Use the folder structure to locate and select the files or folders you want to restore.

### 6. Choose Restore Options
- Select a **restore location**:
  - Original Location *(not recommended for test)*, or
  - **Alternate Location** *(Recommended for testing ‚Äî like Desktop/Test-Restore)*
- Decide whether to **overwrite** existing files (not recommended for test).

### 7. Start the Restore
- Click **Recover**.
- The restore process begins and progress is shown.

---

##  Outcome

Your selected files/folders are restored to the specified location. You can now verify their content to ensure the backup worked correctly.
